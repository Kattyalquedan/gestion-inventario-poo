# Sistema de Gestión de Inventarios

Proyecto desarrollado en Python aplicando Programación Orientada a Objetos.

## Descripción

Este sistema permite gestionar el inventario de una tienda mediante una interfaz en consola. 
Se pueden añadir, eliminar, actualizar, buscar y listar productos.

## Funcionalidades

- Añadir productos validando ID único
- Eliminar productos por ID
- Actualizar cantidad o precio
- Buscar productos por nombre (coincidencia parcial)
- Listar todos los productos registrados
